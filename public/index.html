LOCAL STUDIO

本地运行说明

  1. 演示模式无需配置，所有图片保存在本机，适合体验流程和界面。
  2. 真实生成在启动前设置 OPENAI_API_KEY，服务会自动切换至 OpenAI 图像接口。
  3. 局域网访问使用 python3 server.py --host 0.0.0.0 启动，再通过本机局域网 IP 访问。

真实生成可能产生 API 费用。密钥只从本机环境变量读取，不会发送到浏览器。
